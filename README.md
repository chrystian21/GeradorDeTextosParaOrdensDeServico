📄 Descrição do Projeto

Gerador de textos para ordens de serviço desenvolvido com foco em otimizar e padronizar o atendimento técnico.

Este projeto consiste em uma aplicação que automatiza a criação de descrições para ordens de serviço, permitindo que técnicos gerem textos claros e padronizados a partir de informações básicas como tipo de problema, contato do cliente e horário de atendimento.

A ferramenta foi idealizada para auxiliar no dia a dia de empresas de suporte técnico, reduzindo o tempo gasto com preenchimento manual e evitando inconsistências nos registros.

🎯 Objetivo

Padronizar descrições de ordens de serviço

Reduzir tempo de registro de atendimentos

Evitar erros ou omissão de informações

Melhorar a organização dos chamados técnicos

⚙️ Tecnologias Utilizadas

HTML

CSS

JavaScript

Tauri (para versão desktop)

🚀 Funcionalidades

Geração automática de textos para OS

Templates para diferentes tipos de problema (LOS, equipamento, etc.)

Inclusão de informações opcionais (horário, contato)

Interface simples e rápida para uso no dia a dia

Possibilidade de copiar o texto gerado

💡 Contexto

Projeto desenvolvido como parte de atividade acadêmica no curso de Análise e Desenvolvimento de Sistemas, com foco em aplicação prática para suporte técnico.
